# 🚚 Logistics Operations Dashboard

An interactive logistics analytics dashboard built with Streamlit, Pandas, and Plotly.

## 📊 Features

- Executive KPI cards
- Revenue analytics
- Delivery performance analysis
- Maintenance & fuel cost insights
- Customer revenue analysis
- Interactive sidebar filters

## 🛠 Tools Used

- Python
- Pandas
- Plotly
- Streamlit

## 📈 Key Insights

- Generated over $262M in revenue
- Achieved 85K+ completed trips
- Identified operational bottlenecks through detention analysis
- Analyzed maintenance and fuel spending patterns
